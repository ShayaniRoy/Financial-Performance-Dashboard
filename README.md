Project title
Financial-Performance-Dashboard

Objective:
The goal of this Tableau project is to analyze financial performance across different countries, products, and time periods, 
using key financial metrics such as sales, profit, cost of goods sold (COGS), and discounts.

TechStack: Tableau, Excel

The goal of this Tableau Dashboard Project is to analyze the financial performance of a company using the 
provided Sample Data. 
To create a visually appealing dashboard that provides an overview of the company's financial metrics 
enabling stakeholders to make informed business decisions
